# All-in-one
An all-in-one image with support for multiple programming languages, built for use with Pterodactyl.

- Node.js Latest LTS
  - Bun Latest
  - NVM Latest
  - nodemon
  - PM2
  - pnpm
  - ts-node
  - tsc
  - Yarn
- Python 3.11 & 3.12
- OpenJDK 21 LTS
- Golang
- .NET Core Runtime and SDK
- Puppeteer Support
- Fastfetch
- Utility Software
  - neofetch
  - speedtest-cli
