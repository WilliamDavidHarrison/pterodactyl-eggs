# All-in-one
An all-in-one egg with support for multiple programming languages.

- Popular APT packages (neofetch, speedtest-cli)
- Node.js Latest LTS
  - Bun Latest
  - NVM Latest
  - nodemon
  - PM2
  - pnpm
  - ts-node
  - tsc
  - Yarn
- Python 3.11 & 3.12
- OpenJDK 21 LTS
- Golang
- .NET Core Runtime and SDK
- Puppeteer Support
